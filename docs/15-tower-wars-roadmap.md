# Tower Wars — roadmap (оценка 15.07.2026, панель wf_a917ac85)

Командный слой поверх WOBBLE: **сабреддит = команда**. Глобальный зал славы + сезоны саб-против-саба. Цель — следующий хакатон Reddit (каждые 2–3 мес; правила допускают существенно обновлённые существующие проекты).

## Подтверждённые факты (исследование)

- **Ниша пуста**: 5 галерей прошлых хакатонов — ни одной игры «sub vs sub» (ближайшие: Skyscrabbel — общая башня+рейды, один мир; Pixel Wars — фракции, не сабы; Karma Street — сабы как акции).
- **`redis.global` существует и полноценен** (типы @devvit/redis 0.13.7 + доки: «persist state across subreddit installations»; весь API — zAdd/zRange/incrBy/watch — принимает scope). Внешний бэкенд НЕ нужен. Нюансы: нет листинга ключей (вести реестр инсталляций в hash), квоты global-стораджа не документированы, 5 MB/запрос.
- Пост в чужой саб ненадёжен — и не нужен: война = апп установлен в обоих сабах, каждый постит у себя, координация через redis.global + cron.
- **Бригадинг: легализовать, не запрещать** (урок r/place): рейдер = контент, мемориал именует его И его родной саб; кулдауны 1/3ч ограничивают урон; цель тюнинга — саботаж как специя (10–20% вклада).

## Оценка панели (3 судьи)

unique 7.0 · winChance **5.3** (у WOBBLE-соло было 7.0) · build 8.0 · clarity 8.0 · **lowPop 3.3 (главная беда: войне нужны ДВА живых саба)** · deRisk 5.7.

Шансы следующего хакатона: топ-приз ~12–15% (→30%+ при живой войне на демо), категорийный 25–30%, HM 35–40%, пусто ~25%. Матожидание ≈ как у полировки WOBBLE, дисперсия сильно выше — ставка на хвост.

## Что должно быть правдой

1. **Два живых саба** с реальной историей войны к окну судейства (посев за 3–4 недели, минимум 2–3 дружественных саба).
2. Бригадинг легализован по рецепту выше; неделя живого тюнинга — в бюджете.
3. Сабмишн подан как **новая игра «Tower Wars»** (трейлер с войны, боевые артефакты), не ченджлог WOBBLE.
4. Зал славы самодостаточен и красив при тихих инсталляциях — страховка всей заявки.

## Лестница с гейтами

1. **Спайк redis.global** (0.5–1 д): zSet между двумя реальными инсталляциями. *Не работает → стоп.*
2. **Зал славы** (~3 д): глобальный zSet высот, долгожитель, громчайший обвал недели, реестр инсталляций. Шипается безусловно. *Гейт: <3–5 живых башен в зале → войны заморожены, остаток = polish+промо базы.*
3. **Войны** (~5–8 д): матчмейкинг с активити-гейтом (пара только из сабов с ≥N дропов/день), счёт = рост за окно, нормировка по активным строителям (не подписчикам), баннер «r/A 47:52 r/B», анонсы, флаиры, мемориал рейдера. *Гейт: нет органической пары за неделю → в сабмишне не выпячивать.*
4. **Обкатка саботажа** (1 неделя, живьём).

**Старт — за 5–6 недель до дедлайна следующего хакатона**, иначе сеять войну некогда и потолок — категорийный приз за зал славы.
